Scratch Game: Legit Block Parkour

8/30/2018 - 9/9/2018

The objective of this project was to create a fun and interactive game with my partner using Scratch. During this project my partner and I had a split role where we alternated in designing the game and doing documentation. Some days I worked on documenting the project and other days I worked on coding it. We had a good communication system set in place through texting on the phone for quality teamwork. Although we did have some challenges with coding the sprites in the game. We overcame this with the use of research online about how these certain scratch blocks work. During this project I learned about how important teamwork is because that is how Shravan(my partner) and I created an advanced game in scratch. Using whatever I learn from this I plan to build on my knowledge of the World of Computer Science.
